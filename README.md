# Hospital Management System (MERN Stack)

A comprehensive Hospital Management System built with React, Node.js, Express, and MongoDB. The system features role-based access control for Admins, Doctors, and Staff.

## 🚀 Features
- **Admin Dashboard:** Manage staff, update/delete patient records, and view overall statistics.
- **Doctor Panel:** Access active patient lists, add treatments, and generate patient bills.
- **Staff Portal:** New patient admissions and viewing patient records.
- **Role-Based Security:** Protected routes and JWT-based authentication.
- **Modern UI:** Built with Tailwind CSS and Lucide Icons.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vite, Axios, Lucide React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
